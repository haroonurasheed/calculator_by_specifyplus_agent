
# 🧮 Python CLI Calculator

A simple and interactive **command-line calculator** built using Python.
This project was developed using **SpecPlus framework** for structured design and **Gemini CLI** to assist in code generation and logic improvement.

---

## 🚀 Features

* Basic arithmetic operations (`+`, `-`, `*`, `/`)
* Interactive CLI interface
* Supports continuous calculations
* Simple commands for control

  * `C` → Clear input
  * `quit` → Exit program

---

## ⚙️ Tech Stack

* Python 3
* SpecPlus (project structure & design)
* Gemini CLI (AI-assisted development)

---

## 📦 Project Structure

```
src/
 └── calculator/
      └── cli.py
```

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/python-calculator.git
cd python-calculator
```

### 2. Run the calculator

```bash
python3 -m src.calculator.cli
```

---

## 💡 Usage Example

```
Enter expressions (e.g., '2 + 3 * 4') or 'C' to clear, 'quit' to exit.
> 2 + 8 + 9
```
